Technical Documentation: AI Sync Engine v7.3 + GA4══════════════════════════════════════════════System OverviewThe Event-Driven Intelligence Engine (v7.3) is a high-performance middleware solution built on Google Apps Script. It synchronizes data between HubSpot (CRM), Google Analytics 4 (Behavioral), and MailUp (Marketing Automation).The core value proposition of v7.3 is the Synthetic Signal Module, which uses website engagement data to enhance lead scoring even when traditional CRM signals (emails/meetings) are sparse.🏗 System ArchitectureThe system operates on an 8-Phase Pipeline utilizing a Google Sheet as the central data warehouse and intelligence hub.Ingestion: Real-time webhooks (HubSpot) and scheduled batch pulls (GA4).Aggregation: Signal grouping by unique Lead Email.AI Inference: Generative lead scoring using Gemini 1.5 Flash.Segmentation: Automated tagging and bucketizing.Synchronization: Sequential push to MailUp and bi-directional status updates.🛠 Configuration & Environment VariablesThe system requires specific Script Properties (Extensions > Apps Script > Project Settings) to function:1. Webhook Receiver (doPost)A real-time entry point for HubSpot events.Security: Validates X-HubSpot-Signature-v3.Idempotency: Utilizes a Circular Buffer (PROCESSED_EVENT_IDS) capped at 400 IDs to prevent duplicate processing within a 3-hour window.Concurrency: Employs LockService (10s mutex) to prevent race conditions during simultaneous sheet writes.2. GA4 Synthetic Signal Engine (ingestGA4Signals)Pulls user-level engagement (page views, duration, conversions).Matching Logic: Relies on GA4 customUser:email.Categorization: Automatically converts raw metrics into human-readable subjects like "Deep exploration: 5 pages, 10min".3. Enhanced AI Inference (runAIInferenceEnhanced)Calls the Gemini 1.5 Flash API with a multi-context prompt.Context Injection: Combines HubSpot "Hard" signals with GA4 "Soft" signals.Output: Generates a consistent JSON response: {"score": 1-100, "tag": "one_word"}.4. MailUp Sequential Push (pushSheetToMailUp)Handles the outbound marketing sync.Industry Sorting: Groups leads by Industry to allow for sequential, prioritized pushing.DLQ (Dead Letter Queue): Failed pushes are moved to the Dead Letter Queue sheet for manual review and retry logic.🧪 Production Diagnostic SuiteThe code includes a "Senior Tester" utility to verify pipeline health:diagnoseAIScoringPipeline(): Generates a color-coded status report showing where data bottlenecks occur (e.g., signals ingested but not aggregated).testGA4Connection(): Performs a live handshake with the Analytics API to verify OAuth health.⏱ Trigger Schedule (Recommended)FunctionFrequencyPurposeonEditReal-timePushes manual sheet corrections to HubSpot.syncMailUpToSheetsHourlyPulls Unsubscribes/Bounces from MailUp.syncSheetsToHubSpotEvery 2 HoursUpdates HubSpot lead status based on MailUp data.ingestGA4SignalsDaily (2 AM)Batch imports yesterday's website behavior.fullReconciliationDaily (3 AM)Tri-platform audit to fix missing/stale records.runSyncEvery 6 HoursFull pipeline refresh (Signals → AI → Segments).
